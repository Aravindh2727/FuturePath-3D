# FuturePath 3D 🚀

## 🎯 Career Guidance Platform for Students After 12th

FuturePath 3D is an interactive web application that helps students explore all career options after 12th, including:

- 🎓 Engineering Courses
- 🏥 Medical Courses
- 🎨 Arts & Science
- 💼 Commerce & Management
- ⚖️ Law
- 🌾 Agriculture
- 🛠 Diploma & Polytechnic
- 🔧 Vocational Skills
- 🏛 Government Jobs
- 🏫 Colleges in Tamil Nadu

## ✨ Features

- 🔍 Smart Career Search
- 🧠 AI-based Quiz Recommendation
- 🏫 College Finder (Tamil Nadu)
- 📊 Career Roadmaps
- 📱 Responsive Design
- ⚡ Fast & Modern UI (React + Vite)
